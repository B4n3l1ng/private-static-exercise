Instructions

Follow the instructions below to complete the Space Mission exercise. Make sure to open the developer tools, to check on any potential errors and logs in the terminal.

Iteration 1: Rocket

  • Create a Rocket class.
  • Add a public property name (from constructor argument).
  • Add a private property #fuel (initialized to 100).
  • Add a private method #burnFuel(amount) to decrease fuel.
  • Add a public method launch() that checks fuel before launching. The rocket should only launch, if it has at least 50 fuel. Launching should burnFuel by 50. Depending on the success of failure, should return a string with the shape: "{NAME} has launched into space." or "{NAME} cannot launch due to low fuel"

Iteration 2: Mission Control

  • Create a MissionControl class with a static property rocketsLaunched. Initialize it as 0.
  • Add a static method registerLaunch(rocket) to launch rockets. It should receive the rocket object as an argument, call the launch() method of the rocket object, saving the return in a message variable, log it, and increment the rocketsLaunched static property, if the launch was successful.

Iteration 3: Running the Simulation

  • Create two Rocket objects.
  • Call MissionControl.registerLaunch() for each rocket.
  • Try launching a rocket again to check fuel behavior.
  • Log the total launches using MissionControl.rocketsLaunched.

Expected Output

*** TEST 1: Launch Attempts ***
Falcon 9 has launched into space!
Starship has launched into space!
Falcon 9 cannot launch due to low fuel!

*** TEST 2: Rockets Launched Count ***
Total successful launches: 2
